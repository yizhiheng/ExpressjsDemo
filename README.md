# ExpressjsDemo
A simple web application built with ExpressJS and Redis
### Screenshot
![alt tag](https://raw.github.com/yizhiheng/ExpressjsDemo/master/demo.png)

### Requirements
Nodejs, Redis, npm

### Installation
```
git clone git@github.com:yizhiheng/ExpressjsDemo.git
cd ExpressjsDemo
redis-server
npm start
```

That's it.
